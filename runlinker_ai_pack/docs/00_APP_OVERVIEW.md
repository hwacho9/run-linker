# RunLinker 앱 개요

## 1. 제품 정의
RunLinker는 사용자가 친구 또는 랜덤으로 매칭된 상대와 **실시간으로 함께 달리는 경험**을 제공하는 모바일 앱이다.  
이 앱의 차별점은 단순 러닝 기록이 아니라, **함께 달리는 감각**과 **러닝 종료 후의 공동 결과 정리**에 있다.

## 2. 핵심 가치
- 멀리 있어도 같이 달리는 경험
- 러닝 중 실시간 연결감
- 결과 비교와 동기부여
- 혼자 뛸 때보다 더 지속 가능한 러닝 습관
- 안전한 위치 공유와 프라이버시 기본값

## 3. 핵심 사용자
### 3-1. 친구와 함께 달리고 싶은 사용자
- 서로 다른 지역에 있어도 같이 운동하고 싶다.
- 러닝 약속을 통해 동기부여를 얻는다.

### 3-2. 혼자 뛰기 지루한 사용자
- 랜덤 매칭을 통해 누군가와 같은 세션을 공유하고 싶다.
- 부담은 적되, 실시간 연결감은 원한다.

### 3-3. 기록과 성장을 함께 보고 싶은 사용자
- 누적 거리, 평균 페이스, Sync Score, 함께 달린 비율 등을 보고 싶다.

## 4. 제품 원칙
1. **Home은 실행 우선**  
   앱을 켜면 바로 달리기를 시작할 수 있어야 한다.

2. **Activity는 데이터 중심 허브**  
   세션 기록과 내 통계를 Activity 탭에 모은다.

3. **Friends는 관계 관리**  
   친구 탭은 피드가 아니라 “누구와 달릴지 고르는 화면”이다.

4. **My는 설정/프로필 중심**  
   목표, 프라이버시, 알림, 계정 관리가 중심이다.

5. **Live Run은 감정 중심 화면**  
   사용자는 “지금 진짜 같이 뛰고 있구나”를 느껴야 한다.

6. **랜덤 매칭은 안전이 기본값**  
   정확 위치는 기본 비공개, 시작/종료점 보호, 세션 종료 후 위치 공유 자동 종료.

## 5. 핵심 UX 요소
- Pair View: 두 사람이 함께 달리는 추상 진행 뷰
- Split Live Maps: 내 지도 / 상대 지도 분리
- Sync Bar / Sync Score: 페이스와 진행률 정렬 정도
- Quick Cheer: 짧은 응원 인터랙션
- Activity: 세션 기록 + 내 통계

## 6. 핵심 성공 지표
- 주간 활성 러너 수
- 함께 달린 세션 비율
- 세션 완료율
- 7일 / 30일 재방문율
- 다시 함께 달리기 클릭률
- 친구 초대 전환율
- 랜덤 매칭 성공률
- 평균 Sync Score 확인 비율
