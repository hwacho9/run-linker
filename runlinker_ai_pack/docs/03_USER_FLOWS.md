# RunLinker 사용자 흐름

## 1. 첫 사용자 온보딩 흐름
1. 앱 설치
2. Onboarding 1~3 확인
3. 로그인 / 회원가입
4. 초기 권한 요청
   - 알림
   - 위치
5. Home 진입
6. 첫 러닝 시작

## 2. 친구와 달리기
1. Home 또는 Friends에서 시작
2. Match Setup에서 친구가 선택된 상태
3. 목표 설정
4. 초대 전송
5. 상대 수락
6. Ready Room
7. Live Run
8. Results
9. 친구 유지 / 다음 세션 예약 가능

## 3. 랜덤 매칭
1. Home → 랜덤 매칭
2. Match Setup에서 조건 선택
3. Matching
4. 매칭 성공 시 Ready Room
5. Live Run
6. Results
7. 친구 추가 또는 일회성 종료

## 4. 혼자 달리기
1. Home → 혼자 달리기
2. Match Setup
3. Ready Room 또는 바로 Live Run
4. Results
5. Activity에 저장

## 5. 다시 함께 달리기
1. Activity → Session Detail
2. 다시 함께 달리기
3. Match Setup with previous settings
4. 상대 선택 또는 기존 상대 prefill
5. 새로운 세션 시작

## 6. 데이터 확인 흐름
1. Activity 진입
2. Session History 확인
3. 특정 세션 선택 → Session Detail
4. My Stats 탭 전환
5. 주간/월간 성과 확인

## 7. 프라이버시 설정 흐름
1. My 진입
2. 위치 공유 설정 / 랜덤 매칭 공개 범위 / 기록 공개 범위
3. 설정 저장
4. 이후 Match Setup, Live Run, Results에 반영
