# RunLinker 화면 구성

## 1. Onboarding 1
### 목표
앱의 핵심 가치인 “멀리 있어도 함께 달린다”를 전달한다.

### 주요 요소
- 비주얼 일러스트 또는 그래픽
- 헤드라인
- 짧은 설명
- 다음 버튼

### 핵심 카피 방향
- 혼자 달리는 러닝을 함께 달리는 경험으로

---

## 2. Onboarding 2
### 목표
친구 또는 랜덤 매칭이 가능하다는 점을 설명한다.

### 주요 요소
- 친구 / 랜덤 매칭 설명
- 간단한 카드 UI
- 다음 버튼

---

## 3. Onboarding 3
### 목표
러닝 중 지도/진행률/결과 비교가 가능하다는 점을 설명한다.

### 주요 요소
- Pair View 시각 예시
- 결과 비교 예시
- 시작하기 버튼

---

## 4. Home
### 목표
즉시 러닝을 시작하게 만든다.

### 주요 요소
- 메인 CTA: 같이 달리기 시작
- 보조 CTA: 친구와 달리기 / 랜덤 매칭 / 혼자 달리기
- 요약 카드:
  - 지난 러닝
  - 이번 주 거리
  - 평균 페이스
  - 연속 러닝 일수
- 최근 함께 달린 상대

### 필요한 데이터
- home summary
- 최근 세션
- 이번 주 누적 거리
- streak
- 추천 액션

---

## 5. Match Setup
### 목표
러닝 방식과 목표, 공개 범위를 설정한다.

### 주요 요소
- 모드 선택: friend / random / solo
- 목표 거리
- 목표 시간
- 목표 페이스
- 응원/음성 허용
- 위치 공유 수준

### 모드별 차이
- Friend: 특정 친구가 preselected 될 수 있음
- Random: 공개 범위, 매칭 조건 강조
- Solo: 매칭 관련 옵션 숨김

---

## 6. Matching
### 목표
상대를 찾고 연결 상태를 명확히 보여준다.

### 주요 요소
- 매칭 중 상태
- 후보 러너 카드
- 함께 시작 / 다른 러너 찾기
- 취소 버튼

### 필요한 데이터
- match request status
- candidate list 또는 single candidate
- timeout state

---

## 7. Ready Room
### 목표
달리기 시작 전 두 사람의 목표와 설정을 확인한다.

### 주요 요소
- 내 정보 / 상대 정보
- 목표 거리 / 시간 / 페이스
- 프라이버시 상태
- 준비 완료 버튼
- 카운트다운

---

## 8. Live Run
### 목표
실시간 동행감을 제공한다.

### 주요 요소
- 상단 세션 헤더: 경과 시간 / 목표 / 연결 상태
- Pair View
- Split Live Maps
  - 내 지도 + 궤적
  - 상대 지도 + 궤적
- Dual Stats
  - 거리
  - 현재 페이스
  - 평균 페이스
  - 시간
- Sync Bar
- Quick Cheer
- 제어 버튼: pause / finish / map expand

### 프라이버시 규칙
- Friend mode: exact map 가능
- Random mode: exact map 비노출 기본

---

## 9. Results
### 목표
함께 달린 경험을 공동 성과로 정리한다.

### 주요 요소
- 세션 요약
- 내 지도 카드
- 상대 지도 카드
- 정규화된 비교 뷰
- Sync Score
- 회고 메시지
- CTA:
  - 다시 함께 달리기
  - 친구 추가
  - 공유하기
  - 저장

---

## 10. Activity
### 목표
사용자 데이터의 중심 허브 역할

### 내부 탭/세그먼트
- Session History
- My Stats

---

## 11. Activity > Session History
### 목표
세션 히스토리를 검색/정렬/재진입 가능하게 한다.

### 주요 요소
- 세션 리스트
- 필터
- 정렬
- 각 세션 카드:
  - 날짜
  - 상대
  - 유형(friend/random/solo)
  - 거리
  - 시간
  - 평균 페이스
  - Sync Score
  - 지도 썸네일

---

## 12. Activity > My Stats
### 목표
누적 성장을 보여준다.

### 주요 요소
- 총 거리
- 총 시간
- 총 세션 수
- 평균 페이스
- 최고 페이스
- 주간/월간 그래프
- 혼자 vs 함께 비교
- 가장 많이 달린 친구
- 평균 Sync Score

---

## 13. Session Detail
### 목표
한 번의 세션을 깊게 회고한다.

### 주요 요소
- 세션 요약
- 내 지도 / 상대 지도
- Pair View snapshot
- 구간 페이스 비교
- Sync Score
- 회고 메시지
- 다시 함께 달리기

---

## 14. Friends
### 목표
누구와 달릴지 빠르게 고르게 한다.

### 주요 요소
- 검색
- 필터 칩: 전체 / 지금 가능 / 즐겨찾기 / 최근 함께 달린
- 지금 달릴 수 있는 친구
- 최근 함께 달린 친구
- 전체 친구 리스트
- 친구 추가 / 초대 링크
- empty state

---

## 15. My
### 목표
개인 설정과 계정 관리를 제공한다.

### 주요 요소
- 프로필 카드
- 목표 카드
- 요약 데이터 카드(간단)
- 프라이버시 설정
- 알림 설정
- 차단/신고/계정 관리
- 도움말/문의/약관
